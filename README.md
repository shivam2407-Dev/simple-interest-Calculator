# Simple Interest Calculator

This project is a basic **Simple Interest Calculator** developed as part of a Coursera assignment. It demonstrates fundamental concepts of Git, GitHub, and version control, along with simple financial calculations.

##  Project Description

The Simple Interest Calculator computes interest based on the standard formula:

\[
Simple\ Interest = \frac{Principal \times Rate \times Time}{100}
\]

The program takes user input for:

- Principal amount
- Rate of interest (per year)
- Time period (in years)

It then calculates and displays the simple interest.

##  Features

- Accepts user input
- Calculates simple interest accurately
- Simple and easy-to-understand implementation
- Clean and readable code structure

##  Technologies Used

- Bash scripting
- Git & GitHub for version control

##  Project Purpose

This repository was created to practice:

- Creating and managing repositories
- Branching and merging
- Pull requests
- Using GitHub API with curl
- Writing documentation files (README, CONTRIBUTING, LICENSE)
